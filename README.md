# Ricardo Radtke - Portfolio

Portfolio pessoal de Ricardo Radtke, Filmmaker e Cinematógrafo profissional.

## Tecnologias Utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
